]]> Contact form editor demo

This is a demo of the email form editor available as part of the templateone frontend component suite. It enables Content Managers to create individual "email contact forms" in OpenCms without the need of any programming. The email form editor is also an advanced example of using XML contents in an OpenCms Java application as a mean to generate a powerful front end for the user. The email form editor is currently not available in "Direct Edit" mode.

Please note: To enable this demo (and the rest of OpenCms) to actually send emails you need to configure the email server in the opencms-system.xml configuration file located in the ${OPENCMS_HOME}/WEB-INF/config folder of your servlet container.
