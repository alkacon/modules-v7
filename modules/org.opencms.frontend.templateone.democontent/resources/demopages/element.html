Demo for a page with multiple elements

This is a demo for the templateone "multiple element" feature.

Templateone supports up to 8 content elements one below the other when displaying the accessible layout or 4x2 content elements in the page main body when displaying the classic layout. There are also optional elements that can be inserted on the right or left using a property. Each element can be edited using the "direct edit" feature.

The displayed page version, accessible or classic, can be configured by editing the configuration file "configuration_common" in the templateone configuration folder and changing the "Layout configuration" setting.
